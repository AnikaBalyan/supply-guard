# SupplyGuard AI 🚀

AI-powered platform to predict supply chain disruptions before they happen — detect, analyze, and act with confidence.

## Overview
SupplyGuard AI predicts disruptions using weather, congestion, and delay data.

## Features
- Predict risk levels
- Analyze supply delays
- AI-based explanations
- Action recommendations

## Tech Stack
- Python
- Streamlit
- Pandas
- Scikit-learn

## Demo Features
- Risk prediction based on delay, weather, congestion
- AI-based explanation
- Action recommendations

## How to Run
1. Clone the repository
2. Install dependencies:
   pip install streamlit pandas scikit-learn
3. Run the app:
   streamlit run backend/app.py

## Example Output

CONFIDENCE: 82%  
REASON: Weather + Congestion + Delay history
